# Tripleten Around the U.S.

Around the U.S. é um projeto do programa de Web Development da TripleTen, focado na criação de uma página interativa onde os usuários podem adicionar, remover e curtir fotos; ou editar dados do perfil, como nome, descrição e avatar, caso prefira.

# Principais Recursos Utilizados:

- HTML
- CSS: Flexbox e Grid Layout com base no Figma; Responsividade.
- Javascript: Manipulação do DOM; Validação de Formulário; Objetos e Classes.
- API e Integração de Sistemas
- React + Vite
